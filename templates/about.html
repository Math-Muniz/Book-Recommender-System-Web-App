Book Recommender System Web App

This project is a book recommender system web application created by Matheus Muniz Damasco that allows users to search for books by title. The application is built using Python and the Flask web framework, html and css. The goal of this project is to provide a simple and user-friendly way for users to search for books and find new titles to read. The source code for this project is available on GitHub, and it is open-source, meaning that anyone is free to use, modify, and distribute the code.
